

## React App

React.js App Powered By Redux and Firebase.

### Includes

- React.js
- Redux
- Firebase
- Material-UI
- Redux-Thunk
- Typescript
- Jest

### Running it

```
npm i
npm start
```

### Testing it

```
npm test
```

### Folder structure

    .
    ├── __test__				# unit test cases
    │   ├── components
    ├── src
    |   ├── components        	# reuseable components
    |   ├── config				# app configs
    |   ├── constants
    |   ├── pages
    |   ├── redux
    │   ├── index.js
    └── package.json          	# node dependencies
